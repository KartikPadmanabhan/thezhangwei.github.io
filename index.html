I work with Professor Michael Franz in the Secure Systems and Software Laboratory at the Department of Computer Science. I do research on Programming Languages, Virtual Machines and Compilers.

Projects

ZipPy is a Python 3 implementation built using the Truffle framework.

Substrate VM or SVM is an ahead of time compiled Java Virtual Machine. It tries to use the Graal JIT compiler as the LLVM for Java Apps. Its main target is to hosted language runtimes written in Java and deliver them to the client side in a compact package. This AOT compilation approach greatly reduces start-up time and memory footprint of the hosted program.
